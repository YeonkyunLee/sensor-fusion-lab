🩺 Interactive Surgical Tremor Cancellation

Move your mouse over the canvas — that is the surgeon's intended hand motion. The demo injects a synthetic ~10 Hz physiological hand tremor to make the raw tremulous tool tip, then a real-time adaptive filter (a causal Fourier-Linear-Combiner / adaptive notch) cancels it so the robot tool tip tracks the intended path smoothly. Runs live in your browser, no libraries. From sensor-fusion-lab.

Move the pointer to "operate". Hold still to see the steady-hand hold. Touch/drag works too.

Controls

intended motion (your hand)
raw tip (+ 10 Hz tremor)
cancelled tip (robot)
—
Move the mouse to begin.
Personal side project — fully synthetic demo, no real patient/device data. Honest trade-off: a causal real-time filter can't be zero-lag. Turning adaptation up suppresses more tremor but adds a little tracking lag on fast moves; the FLC notches only the tremor band, so it keeps lag small.
